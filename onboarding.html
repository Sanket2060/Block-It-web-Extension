Installation Steps

  1. 1. Download the Code

    Click on the green button to download the code and get the extension files.

  2. 2. Extract the Zip File and Go to chrome Extensions page

    After downloading, right-click on the zip file and select "Extract Here".Navigate to chrome://extensions in your browser.

  3. 3. Enable Developer Mode

    Toggle the Developer Mode option at the top-right corner.

  4. 4. Load Unpacked Extension

    Click on "Load Unpacked" and select the folder where you extracted the extension files.

  5. 5.Pin the Extension

    Click the Puzzle Button in the top-right corner and pin the extension for easy access.

  6. 6. You're Done!

    You're all set! Start blocking websites and enjoy motivational quotes or memes instead.

  7. For images to use at blocked sites, provide url at extension.
